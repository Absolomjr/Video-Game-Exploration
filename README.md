# Video Game Sales Prediction ML Project

## Overview

This project is a Machine Learning application that analyzes and predicts global video game sales using historical video game sales data. The project demonstrates the complete Machine Learning workflow, including data preprocessing, feature encoding, model training, prediction, and evaluation.

The main objective is to build a regression model capable of predicting `Global_Sales` based on various game-related features such as platform, genre, publisher, and regional sales.

---

## Features

- Data preprocessing and cleaning
- Handling missing values and duplicates
- Encoding categorical features
- Train-test data splitting
- Machine Learning model training
- Sales prediction
- Model evaluation and performance analysis
